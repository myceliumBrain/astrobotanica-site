Primeiro acesso

Ainda não existe nenhum acesso cadastrado. Cole um token de acesso pessoal do GitHub (fine-grained, restrito ao repositório astrobotanica-site, com permissão Contents: Read and write — crie em github.com/settings/personal-access-tokens/new) e escolha uma senha. O token fica criptografado (AES-GCM, chave derivada da sua senha via PBKDF2) e salvo em data/auth.json, no próprio repositório — nunca em texto puro.
